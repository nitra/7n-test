## Огляд

Автоматично оцінює якість документації для заданого набору файлів. Для цього викликає зовнішній скрипт-оцінювач (`docgen-judge-measure.mjs`), передаючи йому список файлів для перевірки та необхідну конфігурацію. Обробляє вихідний код стану, повернутий оцінювачем, для визначення фінального результату оцінки.

## Поведінка

1. Приймає список шляхів до файлів, які необхідно обробити.
2. Якщо список файлів порожній, завершує роботу успішно.
3. Визначає абсолютний шлях до скрипта оцінювача документації (`docgen-judge-measure.mjs`).
4. Запускає скрипт оцінювача, передаючи йому список шляхів до файлів, які потрібно оцінити.
5. Чекає завершення роботи скрипта оцінювача.
6. У разі помилки при запуску процесу, генерує виняток.
7. Завершує роботу з кодом статусу, отриманим від скрипта оцінювача.
