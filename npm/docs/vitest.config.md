## Огляд

Конфігурує процес виконання тестів у проекті. Встановлює параметри запуску та конфігурацію середовища тестування за допомогою Vitest, визначає механізми збору даних про покриття коду та вказує шляхи для збереження звітів про покриття.

## Поведінка

1. Налаштовує конфігурацію для тестування з використанням Vitest.
2. Встановлює постачальника покриття тестами на `v8`.
3. Визначає репортерів для виводу звітів: текстовий, HTML та lcov.
4. Вказує каталог, куди будуть збережені звіти про покриття тестами, відносно каталогу npm.
