## Огляд

Файл забезпечує програмний інтерфейс до AI агента, використовуючи SDK pi. Він підтримує два режими роботи: `callText` повертає текстовий результат від моделі без внесення змін до коду, тоді як `callAgent` активує інструменти кодування та може модифікувати файлову систему у вказаній робочій директорії. Робота з цим файлом є лише для читання (не передбачається пряме написання у файли чи БД).

## Поведінка

callText відправляє запит до AI моделі у текстовому режимі, отримуючи відповідь у вигляді рядка без змін у файловій системі.
callAgent відправляє запит до AI моделі у режимі агента, надаючи повний набір інструментів для роботи з файлами та виконувати зміни у вказаній директорії.

## Публічний API

callText — Надсилає текстовий запит до pi без використання інструментів і повертає відповідь.
callAgent — Надсилає запит до pi у режимі агента з повним набором інструментів для кодування (читання, запис, bash, редагування), при цьому агент безпосередньо створює тестові файли.
