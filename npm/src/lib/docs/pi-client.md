## Огляд

Програмний клієнт забезпечує взаємодію з агентом pi через SDK. Дозволяє отримувати текстові відповіді за допомогою функції `callText` або виконувати автоматичні модифікації файлів у вказаному каталозі через функцію `callAgent`.

## Поведінка

callText відправляє запит до агента pi у текстовому режимі, не використовуючи інструменти, та повертає текстову відповідь.
callAgent відправляє запит до агента pi у режимі кодування з повним набором інструментів, дозволяючи агенту напряму змінювати файли у вказаному робочому каталозі.

## Публічний API

callText — Надсилає одне текстове повідомлення до pi та повертає відповідь, використовуючи конфігурацію з ~/.pi/.
callAgent — Надсилає запит агенту pi з повним набором інструментів для кодування (читання/запис/bash/редагування), причому агент пише тестові файли безпосередньо.
